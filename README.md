This project is showing how a veterinary clinic management system is designed to handle essential operations in  a veterinary practice. It include roles such as the Admin, Vet and Staff, and they all have their own unique functions based on their permissions.

Features:
1. User Authentication: A secure login with role-based access (Admin, Vet, Staff)
2. Pet Management: Add, view, and manage pet details (including, age, medical history, vaccination status).
Linking pets with respective owners.
3. Owner Management: Add and manager owner details(including contact details)
4. Appointment Mangement: Schedule and view appointments
5. Data Storage: Uses text files for storage (pets.txt, ownera.txt, appointments.txt)



How to login:
Admin: username: admin_user  Password: admin_pass
vet: username: vet_user  Password: vet_pass
staff: username: staff_user  Password: staff_pass
